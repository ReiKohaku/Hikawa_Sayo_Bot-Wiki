## 附录：表达式的写法

在某些指令中，您可以通过符合规范的表达式来限定条件。

表达式的组成为：

`<运算符><数值>`

其中，运算符包含以下几种：

| 运算符 | 含义       | 示例 |
| ------ | ---------- | ---- |
| =      | 等于       | =27  |
| !=     | 不等于     | !=29 |
| >      | 大于       | >25  |
| <      | 小于       | <24  |
| >=     | 大于或等于 | >=26 |
| <=     | 小于或等于 | <=28 |
